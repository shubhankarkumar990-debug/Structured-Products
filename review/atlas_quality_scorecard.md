## Dimension Details

### 1. Educational Value (6.0 → 8.0 → 9.5)

**Before B.5:** 2 truncated summaries, 5 vague summaries, 17 generic watch metrics. Building blocks dense without labels. No onboarding guidance.

**After B.5:** All summaries fixed. All watch metrics product-specific. Building block labels added. Start Here section + glossary appendix added.

**After Proposed:** Learning Difficulty appendix would explain WHERE readers get stuck — the highest-value educational addition. Interview Questions appendix adds Socratic learning dimension. Confusion Pairs prevent the most common misunderstandings.

**What moved the score:** B.5 fixed broken content (+2.0). Proposed additions create new learning pathways (+1.5).

### 2. Interview Readiness (5.5 → 8.5 → 9.5)

**Before B.5:** 17 undifferentiated "Watch: Delta" answers. Vague summaries for high-profile products (VarSwap, CDO). No question framework.

**After B.5:** Every watch metric differentiated. All summaries follow ACTION + CLIENT OUTCOME + DISTINCTIVE FEATURE. Glossary explains every metric.

**After Proposed:** Interview Questions appendix provides the exact questions candidates face. Confusion Pairs address "compare X and Y" questions. Learning Difficulty helps candidates anticipate follow-ups.

**What moved the score:** B.5 fixed the data quality gap (+3.0). Proposed additions create interview-specific prep tools (+1.0).

### 3. Reader Onboarding (4.5 → 7.5 → 9.0)

**Before B.5:** No "Start Here" guidance. Dense building blocks. No learning path. New analyst faces 49 cards with no priority ordering.

**After B.5:** Start Here section with 7 anchor products and 3-week learning path. Building block labels. Glossary for technical terms.

**After Proposed:** Learning Difficulty appendix maps the conceptual traps per product — tells the analyst WHERE they'll struggle, not just WHAT they need to learn. Confusion Pairs prevent wrong mental models from forming.

**What moved the score:** B.5 created the onboarding infrastructure (+3.0). Proposed additions deepen it (+1.5).

### 4. Practitioner Utility (8.0 → 8.5 → 8.5)

**Before B.5:** All 16 fields present and accurate. Primary System, ISDA Required, hedge descriptions serve desk practitioners well. Minor: some hedge descriptions expert-only.

**After B.5:** No change to practitioner fields. Glossary helps junior practitioners understand watch metrics.

**After Proposed:** No proposed enhancement targets practitioner utility. This is already well-served. Score unchanged.

**What moved the score:** B.5 added glossary context (+0.5). Proposed adds nothing further (+0.0).

### 5. Publication Readiness (7.5 → 8.5 → 9.0)

**Before B.5:** Structurally complete (3 views, 3 appendices, 49 cards). Two truncated summaries were publication defects.

**After B.5:** Defects fixed. 4 appendices. Updated footer. Clean formatting.

**After Proposed:** Three additional appendices (E, F, G) round out the reference layer. More complete publication for the target audiences.

**What moved the score:** B.5 fixed defects (+1.0). Proposed adds reference depth (+0.5).

### 6. Knowledge Retention (6.5 → 8.0 → 9.0)

**Before B.5:** "Remember These 3 Things" cheat sheet is well-designed. But generic watch metrics meant item #3 was un-memorable for 17 products.

**After B.5:** All cheat sheet items now distinct and memorable. Glossary reinforces technical terms.

**After Proposed:** Interview Questions create retrieval practice hooks — the question triggers recall of the answer. Learning Difficulty identifies the specific concept to anchor.

**What moved the score:** B.5 made all 3 items memorable (+1.5). Proposed adds recall triggers (+1.0).

### 7. Navigation Quality (7.0 → 8.0 → 8.5)

**Before B.5:** 3 views (Family, Complexity, Alphabetical). Section-number-based cards. Table of contents.

**After B.5:** Start Here section provides entry point. Updated ToC with 9 entries.

**After Proposed:** Confusion Pairs appendix provides a new navigation dimension (navigate by similarity/contrast). Minimal improvement since core navigation is strong.

**What moved the score:** B.5 added entry point (+1.0). Proposed adds lateral navigation (+0.5).
