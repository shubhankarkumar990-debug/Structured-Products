# Structured Products — Start Here

**The complete learning and reference library for structured products: one teaching Bible and five companions.**

This page is the front door. It tells you what each document is, who it is for, and when to open it. Every link below works in the digital (HTML) edition; in print, each document is self-contained.

## Quick start by role

- **New joiner, no finance background** → start at the Bible, [Part 0](Desk_Bible_v2.html#part-0), and read forward. Keep the **DNA Atlas** open for recap.
- **New joiner, finance background** → Bible [Part 1](Desk_Bible_v2.html#part-1) → [Part 5](Desk_Bible_v2.html#part-5). Use the **Comparison Matrices** to separate look-alike products.
- **Product control / operations** → Bible [Part 6](Desk_Bible_v2.html#part-6) + the **Infrastructure Encyclopedia**.
- **Trader / structurer** → Bible [Part 5](Desk_Bible_v2.html#part-5) + the **Solutions Manual** scenario library.
- **Interview prep** → Bible product chapters → the **Interview System** → the **Comparison Matrices** for confusion pairs.

## How the library is wired

Every numbered section and Part in the Bible has a **stable anchor** — `#part-6` for Operations, `#sec-5-1-1` for Reverse Convertibles, and so on. References in the text (`[§5.1.1](Desk_Bible_v2.html#sec-5-1-1)`) are clickable and jump straight to the target. An automated **link-integrity check** runs in the build, so these links cannot silently rot.

**Two editions:** the **HTML edition** is searchable and cross-linked across every document — best for daily work. The **print PDF/DOCX** editions are self-contained for offline reading. Same content; only navigation differs.
